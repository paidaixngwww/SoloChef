# 🍳 SoloChef - 一人食智能采购清单

> 专为独居青年设计的膳食管理与智能采购助手  
> **买得精准，吃得从容**

## ✨ 核心功能

### 1. 智能采集 (Smart Capture)
多模态输入，一键生成1人份菜谱：
- 📎 **粘贴链接** - 支持下厨房、B站、小红书等平台
- 📸 **拍照识别** - AI识别餐厅菜品或食谱截图
- ⌨️ **输入菜名** - 直接输入"番茄炒蛋"即可生成

AI自动处理：
- 自动转换为1人份（原500g肉 → 150g）
- 智能取整建议（0.3个洋葱 → 买1个，余下可做洋葱圈）

### 2. 膳食购物车 (Meal Cart)
像电商购物车一样管理你的一周菜谱：
- ✅ 卡片式菜谱展示
- 🎯 勾选本周想吃的3-5道菜
- 📊 实时显示预计食材种类
- 🎨 动态浮窗提示

### 3. 智能归并引擎 (Merge Engine)
处理"多菜一买"的复杂逻辑：
- 🔄 同义词识别（西红柿=番茄，老抽=生抽）
- ⚖️ 单位换算（2个番茄 + 200g番茄 = 500g/3-4个）
- 📂 智能分类：
  - **生鲜** - 需要购买的肉蛋菜
  - **粮油调味** - 检查库存的调味料（可折叠）

### 4. 极简执行清单 (Action List)
超市购物的最佳伙伴：
- ✅ 滑动划掉已购买商品
- 🔍 长按查看食材来源（哪几道菜会用到）
- 📋 一键复制文本（可发给跑腿小哥）
- 📈 进度追踪（5/12 已购买）

## 🎯 核心痛点解决

1. ❌ **问题**: 一个人做饭买菜太难控制份量  
   ✅ **解决**: AI自动转换为1人份，智能取整建议

2. ❌ **问题**: 多道菜怎么合并买菜最划算  
   ✅ **解决**: 智能归并引擎自动合并去重

3. ❌ **问题**: 食材用不完容易浪费  
   ✅ **解决**: 精准计算份量，余料使用建议

## 💡 使用场景

### 周末菜谱规划
1. 打开 SoloChef
2. 浏览收藏的菜谱卡片
3. 勾选本周想吃的5道菜
4. 点击底部浮窗生成采购单

### 超市购物
1. 打开采购清单页面
2. 边逛超市边划掉已买商品
3. 长按查看食材用途
4. 一键复制发给朋友

### 添加新菜谱
1. 看到喜欢的菜
2. 拍照/粘贴链接/输入菜名
3. AI自动处理为1人份
4. 自动保存到菜谱库
