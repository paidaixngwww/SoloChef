# SoloChef - 功能演示说明

## 🎯 核心流程

### 1️⃣ 首页 (HomePage)
**路径**: `/`

**功能**:
- 显示所有保存的菜谱卡片（2列网格布局）
- 顶部统计栏显示：总菜谱数 / 已选择数 / 预计食材种数
- 每个菜谱卡片包含：
  - 菜品图片（带渐变遮罩）
  - 菜名、食材数量、人份
  - 标签（快手、健康、家常等）
  - 选择复选框（点击选中/取消，阻止卡片点击）
  - 删除按钮（悬停时显示，阻止卡片点击）
  - **点击卡片主体跳转到详情页**
- 右上角按钮：
  - 绿色问号按钮 → 跳转到关于页面
  - 红色加号按钮 → 跳转到添加页面（空状态时有脉冲动画）
- 底部浮窗（选择菜谱后自动显示）：
  - 显示已选菜数和预计食材种类
  - 点击"生成采购单"跳转到购物清单页

**交互细节**:
- 卡片悬停时轻微上浮（-4px）+ 阴影加深 + 显示"查看详情 →"
- 复选框勾选时有缩放动画（阻止卡片点击事件）
- 删除按钮悬停显示（阻止卡片点击事件）
- 底部浮窗滑入动画 + 箭头呼吸动画
- 空状态展示欢迎界面

---

### 2️⃣ 菜谱详情页 (RecipeDetailPage)
**路径**: `/recipe/:id`

**功能**:
- 顶部导航栏：
  - 返回按钮（返回首页）
  - 加入购物车按钮（切换选择状态）
    - 未选择时显示"加入购物车"（橙色）
    - 已选择时显示"已选择"（绿色+对勾）
- 大图展示（带渐变遮罩）
- 菜谱信息卡片：
  - 菜名（大标题）
  - 人份数和食材种类数
  - 标签列表
- 食材清单（分类显示）：
  - **生鲜食材** 🥬
    - 所有 category='fresh' 的食材
    - 每项显示：食材名 + 用量标签
  - **调味料** 🧂
    - 所有 category='pantry' 的食材
    - 每项显示：食材名 + 用量标签
- AI 提示卡片：
  - 说明已转换为1人份
  - 提示可在购物清单查看合并用量

**交互细节**:
- 页面元素分层淡入（图片→信息→食材列表）
- 食材项从左侧依次滑入
- 加入购物车按钮有缩放动画和颜色切换
- 导航栏固定在顶部（滚动时保持可见）

---

### 3️⃣ 添加菜谱页 (AddRecipePage)
**路径**: `/add`

**功能**:
- 三种输入方式选择：
  1. **粘贴链接** (橙色)
     - 输入下厨房/B站/小红书链接
     - AI提取菜谱信息
  2. **拍照识别** (绿色)
     - 上传菜品照片
     - AI识别菜名和食材
  3. **输入菜名** (黄色)
     - 直接输入"番茄炒蛋"
     - AI生成标准1人份清单

**交互细节**:
- 模式卡片点击时有缩放动画
- 选择模式后淡入显示输入区域
- AI处理时显示加载动画（Sparkles图标旋转）
- 处理完成后自动返回首页

**AI预处理说明**:
- 自动转换为1人份
- 智能取整（0.3个洋葱 → 建议买1个）
- 同义词识别
- 单位标准化

---

### 4️⃣ 采购清单页 (ShoppingListPage)
**路径**: `/shopping-list`

**功能**:
- 进入时显示2秒AI归并动画
- 食材分两类显示：
  1. **生鲜食材** - 需要购买（默认展开）
     - 肉、菜、蛋、奶等
     - 每项显示用量和使用菜谱数
  2. **粮油调味** - 检查库存（默认折叠）
     - 油、盐、酱油等调味料
     - 点击展开/折叠

**单个食材项包含**:
- 复选框（点击勾选/取消）
- 食材名 + 用量
- 使用道数提示
- 信息按钮（长按查看来源）

**顶部功能**:
- 显示购买进度（5/12 已购买）
- 复制按钮（一键复制纯文本清单）

**交互细节**:
- 食材项从左侧滑入动画
- 勾选后置灰 + 删除线
- 复选框勾选时缩放动画
- 点击信息按钮弹出底部抽屉
  - 显示该食材用于哪几道菜
  - 提示"记得买够份量"

## 🎨 设计系统

### 配色方案
```css
--cream-white: #F9F7F2;    /* 奶油白背景 */
--persimmon-red: #E07A5F;  /* 柿子红主色 */
--sage-green: #81B29A;     /* 鼠尾草绿辅色 */
--charcoal: #3D405B;       /* 深炭灰文字 */
--tangerine: #F4A261;      /* 橘黄强调色 */
```

### 圆角规范
- 卡片容器: `16px` (rounded-2xl)
- 按钮: `12px` (rounded-xl)
- 小标签: `8px` (rounded-lg)
- 圆形按钮: `50%` (rounded-full)

### 间距规范
- 页面边距: `24px` (px-6)
- 卡片间距: `24px` (gap-6)
- 元素内边距: `16px` (p-4)
- 小间距: `8px` (gap-2)

### 阴影层级
- 默认: `shadow-sm` (轻微阴影)
- 悬停: `shadow-md` (中等阴影)
- 浮窗: `shadow-lg` (较强阴影)
- 模态: `shadow-2xl` (最强阴影)

## 🚀 交互动画

### 页面转场
- 进入: `opacity 0→1` + `scale 0.9→1`
- 退出: `opacity 1→0` + `scale 1→0.9`
- 时长: `200ms`

### 悬停效果
- 按钮: `scale 1→1.05` (hover) + `scale 1→0.95` (active)
- 卡片: `translateY 0→-4px` + 阴影加深
- 时长: `200ms`

### 状态反馈
- 复选框: `scale 0→1` (勾选时)
- 列表项: `translateX -20→0` (进入时)
- 浮窗: `translateY 100→0` (滑入时)

### 持续动画
- 加载图标: 旋转 `360deg`
- 箭头提示: 左右移动 `0→5→0px`
- 食材汇聚: 上下浮动 `0→-10→0px`

## 📱 响应式设计

### 断点
- 移动端: `< 768px` - 单列布局
- 桌面端: `≥ 768px` - 双列布局

### 适配策略
- 菜谱卡片: 移动端1列，桌面端2列
- 统计栏: 3个统计项始终横向排列
- 底部浮窗: 全宽显示，左右留边距
- 文字: 响应式缩放，保持可读性

## 🎯 核心算法

### 食材归并逻辑
```typescript
1. 遍历所有已选菜谱
2. 提取每道菜的食材列表
3. 同义词标准化（西红柿→番茄）
4. 按食材名称分组聚合
5. 累加相同食材的用量
6. 记录每个食材的来源菜谱
7. 按类别分类（生鲜/粮油）
8. 按类别+名称排序
9. 返回合并后的清单
```

### 同义词映射表
```typescript
{
  '西红柿': '番茄',
  '土豆': '马铃薯',
  '洋葱': '圆葱',
  '老抽': '酱油',
  '生抽': '酱油'
}
```
